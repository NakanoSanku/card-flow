AugmentCode 提供的官方 MCP Context Service，使支持 Model Context Protocol（MCP） 的客户端可以直接接入 AugmentCode 的代码索引与上下文服务，用于在 IDE 或 Agent 中检索仓库、符号和代码片段等信息，提升代码理解与导航效率。
